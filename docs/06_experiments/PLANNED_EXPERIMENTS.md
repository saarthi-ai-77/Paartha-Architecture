## (Never executed) EXP-001: Primitive Isolation Test
* **Objective:** To determine if a singular, specialized computational primitive (e.g., a spatial reasoning primitive) outperforms a generalized LLM on a strict structural composition task.
* **Hypothesis:** HYP-001 (The Multi-Primitive Hypothesis).
* **Method:** Develop a highly constrained toy environment requiring spatial layout rules. Compare the success rate of a hand-crafted "spatial primitive" against a prompt-engineered foundation model.
* **Expected Outcome:** The specialized primitive will demonstrate 100% adherence to layout constraints, while the foundation model will exhibit probabilistic failures.
* **Status:** Never executed. Superseded by the build-experiment-validate track (DEC-005) before it was run.
