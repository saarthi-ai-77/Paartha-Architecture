# Experiment Templates

## Standard Experiment Protocol

When designing an experiment to test an architectural primitive or hypothesis, use the following template. Do not fabricate results; clearly mark planned experiments.

```markdown
# Experiment: [Name/Identifier]

## Objective
[What specific architectural component or hypothesis are we testing?]

## Hypothesis
[Reference to a specific HYP-XXX from the Hypothesis Register]

## Method
[Detailed description of the experimental setup, evaluation domain, and execution steps]

## Expected Outcome
[What we anticipate will happen if the hypothesis is true]

## Actual Outcome
[To be filled post-execution. Raw data and observations]

## Conclusion
[Analysis of the actual outcome against the expected outcome]

## Future Work
[How this experiment informs the next steps in the roadmap]
```
